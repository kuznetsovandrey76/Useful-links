Welcome to HELP page.

  1. В пустой дериктории virtualenv env
  2. cd env/Scripts
  3. activate
  4. cd ../..
  5. pip install flask
  6. pip install gunicorn
  7. touch Procfile
  8. В Procfile записываем web: gunicorn project:app, где project - название основного файла в директории (ex: project.py)
  9. pip freeze > requirements.txt
  10. Заливаем в Heroku
  11. git push heroku master
